The Problem

V1 rocks symmetrically — and that's a problem

The original Flow-Motion uses a flat leaf spring running parallel to the ground. It provides a great rocking motion — but with equal force in both directions, three issues emerge:

⚠ Pinch Points
Equal forward rock means the front gap closes under load — creating potential finger/toe pinch points that require guards or design compromises.
One-Size Rocking
Every chair model gets the same rocking feel. No way to tune for different seat heights, weight ranges, or use cases.
Static Appearance
A flat spring parallel to the ground reads as inert. It doesn't communicate "this chair rocks" until someone sits in it.
The Solution

Angle the spring. Change everything.

V2 places a wedge component beneath the leaf spring, tilting it at an angle. This single change creates two distinct fulcrum points with different lever arms:

🔴 Rear Fulcrum (Fixed)
Where the spring meets the back of the wedge. Shorter lever arm = less force to rock backward. The "easy lean-back."
🔵 Forward Fulcrum (Tunable)
Position along the spring is adjustable. Controls how hard it is to rock forward, how far forward it goes, and whether the front gap ever closes.
Key Insight

One parameter. Three outcomes.

The forward fulcrum position is the single design variable. Slide it forward or back to simultaneously control:

⚖️
Force Asymmetry
How much easier backward rocking is vs. forward. Dial in the natural "lean back, gentle return" feel that people prefer.
📏
Forward Amplitude
How far forward the chair can travel. Fulcrum all the way forward = zero forward rock. Move it back = progressively more travel.
🛡️
Pinch Gap
Minimum front gap at max deflection. Tune to always stay above the safety threshold — pinch points eliminated at the mechanism level.
Design Implication
This isn't just a spring change. It's a tuning system. Same base frame, same leaf spring — swap the wedge to customize the rocking feel for every chair in the lineup.

Leaf Spring
Flat steel — same proven material as V1. No new spring tooling required to prototype the V2 concept.
Wedge Component
Separate piece from base frame. Can be injection molded plastic or stamped steel. Cheap tooling, easy to iterate. Acts as the "tuning cartridge" for the system.
Visual Impact
The angled spring reads as motion even at rest. Communicates "this thing rocks" before anyone sits in it. V1's flat spring looks static by comparison.
Platform Advantage

One system. Entire lineup.

🔧 Standardized Base
Same base frame tube and leaf spring across every Flow-Motion chair. Reduces tooling cost, simplifies manufacturing, lowers SKU complexity.
🎛️ Tunable Per Model
Different wedge = different feel. A compact camp chair gets a different rocking profile than a tall director's chair. One tooling change, not a frame redesign.
💰 Low Tooling Cost
Wedge is a simple molded or stamped component. Cheap to tool, cheap to modify, cheap to test multiple variants. Way cheaper than bending new frame tubes.
🏪 Retail Flexibility
If a retailer wants a different "feel" for their exclusive variant, it's a wedge swap — not a frame change. Customize without multiplying tooling investment.
The Wedge Is the Personality
Base frame, spring, and chair structure stay standardized. All the rocking personality lives in one small, cheap, swappable component.
