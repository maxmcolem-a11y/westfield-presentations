Height-Adjustable Outdoor Table

A general-purpose outdoor table platform built from known, reliable fold-flat construction.

Positioned for coffee height, standard table height, or bar height — with a practical price point and broad channel fit.

Westfield Outdoors · Platform Concept
01 / 09

Lead With the Table, Not the Origin Story

The original Camp / Field Utility Board concept is useful, but the bigger opportunity is simpler: a portable outdoor table that changes height.

It can serve as a low coffee table, a normal utility table, or a standing / bar-height surface without asking the consumer to understand a niche use case first.

Low

Coffee table / lounge height

Mid

Standard prep and dining height

High

Bar, standing, or serving height

02 / 09

A Familiar Frame With a Bigger Job

The winning idea is not a novelty product. It is a hard-top, adjustable-height, fold-flat table for everyday outdoor use.

  • Backyard entertaining
  • Camping and RV use
  • Beach, patio, and park days
  • Tailgate serving
  • Garden, fish, hunt, and utility work
03 / 09

Why the Construction Makes Sense

The value is in using construction we already understand: metal tube legs, simple locks, compact folding geometry, and a hard outdoor surface.

That reliability matters because it lets the product feel credible without driving it into premium-price territory.

  • Known mechanism logic
  • Low-complexity components
  • Efficient folded storage
  • Durable outdoor material story
  • Clear VA / VE path for mass retail
04 / 09

One Product, Three Height Stories

The adjustment range is the core selling feature. It gives one SKU multiple buying reasons without adding extra accessories.

Coffee Height

Low lounge table for chairs, blankets, fire pits, and patios.

Standard Height

Everyday table for prep, serving, games, tools, and camp work.

Bar Height

Standing-height surface for tailgates, parties, drink stations, and quick service.

05 / 09

Broader Than Camp Utility

“Camp / Field Utility Board” is a strong internal development name, but the market-facing story should be broader.

Consumers understand outdoor tables immediately. The height adjustment gives them the reason to upgrade.

  • Outdoor side table
  • Portable prep table
  • Serving table
  • Standing social table
  • Wagon-side or chair-side work surface
06 / 09

Platform Potential

If the base mechanism performs, this does not need to stop at one table size.

The same construction can support a family of outdoor tables across price points and use cases.

  • Chair-side mini table
  • Low lounge table
  • Standard camp / prep table
  • Tall bar-height serving table
  • Wagon-compatible deployable surface
07 / 09

What We Need To Prove

The next step is not more storytelling. It is a fast prototype that proves the height range, stability, lock confidence, and cost structure.

01Stable at low, standard, and tall positions
02Does not wobble under real side loads
03Feet work on grass, gravel, patios, and dirt
04Lock feels obvious and safe
05Folded profile stores cleanly in garage, car, RV, or wagon
06Bill of materials supports an attractive retail price
08 / 09

Recommendation

Build the Outdoor Table First

Fork the Camp / Field Utility Board concept into a cleaner, broader product: a height-adjustable outdoor table with reliable construction and approachable pricing.

This gives Westfield a simple sell-in story, a visible feature advantage, and a mechanism platform that can scale into a family.
