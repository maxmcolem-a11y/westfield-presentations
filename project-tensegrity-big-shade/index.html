Concept paths
05
Two serious options

The real choice is where the deployment intelligence lives.

Pure tensegrity

Structure carries the intelligence

  • Pre-rigged cables, self-locking members, no field tuning
  • Best brand story and cleanest structural read
  • Only viable if lock state is obvious and forgiving
Guided hybrid

Guide system carries the intelligence

  • Uses ribs, hinges, or guided geometry to simplify the routine
  • Faster path to calm shade and membrane integration
  • Fails if it turns into ordinary deployable hardware
Pure wins if
hard-stop geometry replaces tuning.
Pure loses if
the user still has to trust invisible math.
Hybrid wins if
it gets to stable shade faster without losing identity.
Hybrid loses if
the tensegrity read disappears.
Recommendation
06
Best current call
Lead with the hybrid. Keep pure tensegrity alive.

The hybrid is the better product-development path today because it can prove footprint, setup clarity, and wind calmness faster. Pure tensegrity should stay alive as the higher-upside mechanism track.

Lead

Get to a believable pavilion quickly and prove real user experience.

Preserve

Keep pure mechanism work wherever it buys meaningful mass, packed-size, or identity gains.

Avoid

Do not drift into either a generic event canopy or a fragile engineering demo.

07
One-person storyboard

The routine has to be readable before it can be beautiful.

01 · Carry

One clear bundle

The packed state should look intentional and manageable.

02 · Open

Let the frame find itself

The geometry should begin locating itself before the membrane enters the story.

03 · Lock

Confirm the end state

The operator needs a visible yes signal, not a guess.

04 · Shade

Add the membrane last

Once the frame is calm, the fabric turns mechanism into place.
